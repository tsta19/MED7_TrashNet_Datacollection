# TrAAUsh - A data set of litter in context
An image dataset of litter in context that has been collected and annotated as part of several student projects at [Aalborg University (AAU)](https://www.aau.dk/) in the Departmen of [Architecture, Design and Media Technology](https://www.create.aau.dk/). The dataset was collected in Denmark only (to the best of our knoweldge). 

If you want to get involved please contact [Markus Löchtefeld](https://vbn.aau.dk/da/persons/137170)


## Data Collection
The images were collected through the SpotAffald app which is available both in the [Google Play](https://play.google.com/store/apps/details?id=dk.ChatrineChristoffer.med10_msc) Store and Apple [AppStore](https://apps.apple.com/dk/app/spotaffald/id1567051438?l=da). The app was developed as part of a [Master Thesis](https://projekter.aau.dk/projekter/en/studentthesis/spotaffald-a-citizen-science-approach-to-litter-in-context(19a17742-7ff5-4f0b-9de2-93c8e3b68dc8).html) by Christoffer Cæsar Fælled and Chatrine Elisabeth Larsen.
